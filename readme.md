# Market Street Tycoon

## A js13kgames game

![screenshot from the game](thumb_small.png)

Become the most successful merchant in the kingdom!

Market Street Tycoon is a game about building up a successful business in mediaeval Europe, selling a variety of products for the best price. The more you sell your products, the more your reputation increases. This unlocks new possibilities, and your costumers are more willing to pay you more for your quality goods. ## Development

Install dependencies

```sh
yarn
```

Run development

```sh
yarn dev
```

Build a new version

```sh
sudo apt install default-jre # Needed to run roadroller
yarn build-with-best-roadroller
```
